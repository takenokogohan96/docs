## イベント参加動機

### 100傑のリベンジ 
![star_impression.png](/img/user/img/Personal/star_impression.png)
 自分がミリシタを始めてから数えて6度目の楽曲付きイベントとなる「Star Impression」でのことでした。

ノリと勢いだけで100傑を目指していましたが、3日目105位まで進んだところで石が足らず最後まで全速力で走れませんでした。当時は諸事情で副業をしておらず完全な手詰まりです。

やる気はあったものの泣く泣くリタイア。[^1][^2]経済的安定を手に入れて必ずTPRをリベンジをすると誓いました。（直後に副業を再開）
## キービュアルセンターの大抜擢
![7days_smile.png](/img/user/img/Personal/7days_smile.png)
恵美（以下担当）は先頭から一歩引いて周りを引き立たせる動きをしがちなタイプだと個人的に解釈しています。そんな担当が周年の大舞台で先頭に立ち堂々としている…何らかの強い覚悟があったに違いないとキービジュアルから感じ取りました。

ここでリベンジを兼ねての100傑を目指したいという気持ちがかなり強くなりました。が、他のPが綴った周年イベのnoteなど見ていると相応の準備と覚悟を決めて臨んでいる方が多く、これだけの熱量を持ったPと同じ土俵で戦えるだけのマインドがあるものか…と引き下がり始めていました。

「始めてからまだ1年弱だし、周年100傑には早い」と、謎のブレーキをかけ始めた自分は100傑に拘らずゆるりと周年を過ごすつもりでした。
### メインコミュ第145話「I believe...」
![beautiful_believer.png](/img/user/img/Personal/beautiful_believer.png)
自分はメインコミュ第70話「Dears」がきっかけで恵美担当になろうと決意した経緯があります。このコミュを初めて観たとき浮かんだゆるい疑問「どうしたらこの子を幸せにできるのか」と、その答えを漠然と考えながら1年弱担当を続けてきましたが、今回のコミュ[「I believe...」](https://x.com/imasml_theater/status/1799637546890142100)でなんと担当は自らを受け入れ変わり始めていたのです。

自分も競争の類は苦手です。が、周年センターに続けてメインコミュ・SFYと続けて恵美の強い想いに触れるごとにはっきりと意識の変化がありました。担当は自分を信じて変わろうとしているのに、自分は為しえようとする前に諦め始めている、なんて愚かなことかと…ここで引いたらきっと後悔する。

ここで気持ちに整理がつき、恵美100傑を目指す準備に入りました。
## イベント目標
[昨年度周年イベント【R@ise a Good Sign!】](https://mltd.matsurihi.me/events/290)のキービジュアルセンター、望月杏奈の100位「4,502,677pt」を少し上回る数値の「5,000,000pt」としました。
また事前に進捗管理表を作成し、イベント序盤で速度が足らないと判断した場合は100傑狙いを降りる想定で挑みました。
## 環境と準備
イベントを走ると決意を固めたのが6月中旬と遅れたこともあり十分な準備が行えなかった点がありました。
### 稼働時間
イベント3週間前にして2日だけ有給を入れることができ、仕事に支障が出ないギリギリまで睡眠時間を削る計画を立て合計 174h の稼働時間を確保しました。
### 端末とNWインフラ
ミリシタを始めてから端末は [Xperia PRO](https://www.gsmarena.com/sony_xperia_pro-10707.php) を使用していましたが、5月に [Galaxy S23 Ultra](https://www.gsmarena.com/samsung_galaxy_s23_ultra-12024.php) を導入したためよりパフォーマンスが高いそちらを利用しました。
自宅回線の minひかり と RTX1210 + WLX402、IIJmio の D回線は普段使いのものをそのまま使いました。
### ゲーム内アイテムの備蓄
イベント開始直前の各原資の数は以下の通り

| 原資   | 数量             |
| ---- | -------------- |
| 石    | 9,897          |
| ドリンク | 10が3つ<br>20が1つ |
| オーパス | 約50枚           |
100傑を狙いに行く備蓄量でないことは素人の自分ですら分かる状態でしたが、要は石さえ尽きなければ走り続けられると判断しました。
### その他
買い物の時間を発生させたくなかったので約2週間分の食料は事前に書き込みました。（ベーブブレッド、MAXVCOFFEEなど）
あとは予め部屋をお掃除し、開始時刻00:00に合わせて仮眠しいざ開始です。
## イベント中のハイライト
### Day1
- 夜間9時ごろ。ボーダーbotに2~3位で名前が載る瞬間を目撃[^3]
- Day1終了時点で4位につけ、S23Uへの信頼度が上がる[^4]
### Day4
- 過去イベントの因縁「Star Impression」で原資を獲得。手打ちし続けた楽曲の中では最も高いモチベで続けられた
### Day7
- 当初23時間走る予定だったものの、インゲーム中に寝落ちしてしまい何度も石を払ってリスタートする。疲労がピークに達し21時間稼働でリフレッシュタイムへ突入
### Day11
- 完全にオーパスが枯渇する。可能な限り手打ちをしていたが ilLUmiNAte! を手打ちし続けるのは厳しく、効率低下覚悟で バトンタッチ も随所に挟んだ
- この日あたりから最終的な着地点が「5,000,000pt ～ 5,500,000pt」くらいになると予測
### Day12
- この日も当初23時間走る予定だったものの、やはり疲労がピークに達し19時間稼働でリフレッシュタイムへ突入
- この時点でスタミナ（＝石）の消費量も既に想定を超えていた。出費が日常生活に影響を与えるラインに達し始めたためリフレッシュタイム中に無償石堀りを実施
### Day13
- 午前の3時に起床するつもりが目覚ましの設定を忘れたまま爆睡。たまたま目覚ましを6時に設定していたこともあり3時間ロスで済む
- **イベントpt累計5,000,000を突破（目標達成）**
- イベント残り3時間あたりからProduce Noteさんの[PST ポイント調整](https://produce-note.bknxyrg.com/mltd/pst-point-adjustment/theater)を使い着地点を定める
- イベント終了の15分前に目標だった500万+サンキューの語呂合わせで**「5,390,000pt」**に着地し終了

## 感想
スパークドリンクも石もオーパスも使いつぶしましたが無事に目標達成して素直にうれしい気持ちです。「Star Impression」でのリベンジも晴らすことができ、担当の想いにも呼応できたのではないかと自負しています。

今回100傑狙いにしては明らかにオーバーランなポイントまで走りきったことで10傑の間に存在する見えない「壁」を認識することもでき、もっと入念に下準備すれば十分に超えられる壁なのではないかと感じました。
